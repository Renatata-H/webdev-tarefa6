    Aula 1

    Nos ensinaram a diferença entre Internet (a rede) e Web (a coisa que a gente usa quando acessa a rede). Por isso é WebDev, e não InternetDev, e não só porque rima. Enfim, sobre HTML: no explicaram que significa Hyper Text Markup Language. Markup language, porque ela usa tags usando "<" e ">" para identificar coisas; hyper text porque ela conecta a página a outras páginas. O glorioso HTML faz parte da tríplice trindade da Web, ao lado do CSS e do Javascript - é como se HTML faz uma coisa, o CSS deixa a coisa muito legal ao usuário e o JS faz a coisa funcionar. A vida da gente, claramente, foi escrita apenas em HTML. Estruturalmente, o HTML, funciona com uma "head", onde estão os metadados da página, como título, cores, etc., e um "body", onde está tudo o que se exibe ao usuário. Por fim, demonstraram uma lista de tags para a gente :). Eis algumas: br/, h1, h2, h3, h4, h5, h6, id="?", p, div, button, a href="link", img, ol, ul e li.

    Aula 2

    A gente aprendeu o CSS hoje - isto é, o modo "Cansei de Ser Sexy" do site. O CSS deixa o site bonito, em suma... A chave para usá-lo é deixar as coisas bem marcadinhas no HTML, com ID's e classes, pois o CSS só consegue personalizar coisas se você disser o que exatamente você quer personalizar. O CSS faz parte da tríplice trindade da Web, ela deixa as coisas que HTML fez parecerem mais agráveis. Estruturalmente, no CSS basta chamar um seletor, como uma tag, um ID ou uma class, e escrever dentro de chaves ("{}") tudo o que se precisa para fazer seus olhos não saltarem das suas órbitas.

    Aula 3

    Nos ensinaram sobre JavaScript, o último da trindade Web. Apesar de ser o mais competente entre os três (só esse campeão que é uma linguagem de programação de fato), ele é o irmão esquisito entre as linguagens... Ele genuinamente acha que dividir infinito por infinito dá infinito, tadinho. O JS é todo esquisito na sua estrutura, os AND (&&) e OR (||) dele só funcionam certo se estiverem dentro de if's, caso contrário eles são os próprios if's. Apesar disso, ele é bem útil. Sem ele, o site faz nada, que nem esse site aqui. Inútil.
